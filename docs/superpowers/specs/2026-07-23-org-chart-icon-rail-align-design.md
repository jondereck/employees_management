# Org Chart Icon Rail + Horizontal Align Design

## Goal

Make the canvas tools feel Lucidchart-like and keep peer connections level.

## Approved decisions

1. **Icon rail (Option A):** Narrow vertical tool rail on the left edge of the canvas. Keep the existing Offices / Quick Add / View Options left panel.
2. **Fullscreen exit:** Keep Exit button; make Esc reliably exit fullscreen; show a short Esc hint in the fullscreen toolbar.
3. **Horizontal align:** When a connection is mostly horizontal (or a horizontal T-bar), snap related nodes to a shared Y so the line is level (not diagonal).

## Out of scope

- Replacing the Offices panel with flyouts
- Full Lucidchart feature parity (shapes library, freehand, etc.)
